# 順記燒臘館 · 使用說明

---

# 🔴 最重要：貼在 LINE 裡的網址，要加一個尾巴

以後你在 **LINE 社群 / 聊天室 / 群組**貼網站連結，**一律用這一條**：

```
https://wsad952701-beep.github.io/123/13/?openExternalBrowser=1
```

**`?openExternalBrowser=1` 這個尾巴不能少。**

### 為什麼？

LINE 的內建瀏覽器**沒辦法叫起 LINE App**（等於叫自己），所以客人在 LINE 裡開網頁，
點「加入社群」就會卡在 **「請啟動 LINE 應用程式」** —— 就是你遇到的那個畫面。

加上這個尾巴，LINE 就會改用手機的 **Safari / Chrome** 開網頁。
在那裡點社群連結，才叫得起 LINE App，一切正常。

> 💡 網頁本身也做了自動偵測：萬一客人用舊網址進來，會**自動幫他彈到瀏覽器**。
> 但你貼新網址最保險，少一次跳轉。

> 💡 如果自動彈出失敗（很舊版的 LINE），網頁底部會跳出綠色提示條，
> 教客人「點右上角『⋯』→ 用預設瀏覽器開啟」，還有一鍵開啟的按鈕。

# ❓ 點了「加入 LINE 社群」沒反應？

## 先確認：你是在哪裡開這個網頁？

| 在哪開的 | 會怎樣 |
|---|---|
| **Safari / Chrome** ✅ | 點下去 → 直接跳出 LINE App → 加入社群 |
| **LINE 的內建瀏覽器** ❌ | LINE 叫不起自己 → 卡在「請啟動 LINE 應用程式」 |

> 怎麼分辨？畫面**最上面有 ✕ 和 ⋯** 的，就是 LINE 的內建瀏覽器。
> 網頁會自動幫你彈出去；如果沒彈成功，底部會出現綠色提示條教你手動開。

## 還是不行？（iOS 常見）

iPhone 有個機制：**如果你曾經選過「在 Safari 中開啟」，iOS 就會記住，
之後再點同一個網域就不會叫起 App 了。**

解法：

> **長按**綠色的「加入 LINE 社群」按鈕 → 選 **「在『LINE』中開啟」**

或用網頁上的備援：

> 點閘門裡的 **「沒跳出 LINE？點這裡」** →
> **「複製社群連結」** → 打開 LINE → 貼到搜尋列 → 加入

# ⭐ 進階（選配）：拿到客人真實的 LINE 名稱

**不做也沒關係，網站照樣運作。**

做了之後：客人加入社群之後，會**再跳出一次 LINE 官方授權畫面**，
按「允許」才能點餐。你就能拿到他**真實的 LINE 名稱**，訂單上會多一行：

```
LINE　　｜林小美          ← 一看就知道是誰下的單
```

好處：擋得住亂填名字的人，而且是**LINE 官方驗證過的身分**。

### ⚠️ 完全不需要 LINE 官方帳號

用你**自己平常在用的 LINE 帳號**登入開發者後台就好。
**免費**、不用申請官方帳號、不用營業登記、不用綁信用卡。

### 怎麼做（手機就能做，約 5 分鐘）

**1. 打開 <https://developers.line.biz/console/>**
> 點【LINE 帳號登入】→ 用你自己的 LINE 掃碼登入
> 第一次會叫你同意開發者條款，按同意。

**2. Create a new provider**
> 名稱填 `順記燒臘館` → **Create**

**3. Create a new channel → 選【LINE Login】**
> | 欄位 | 填什麼 |
> |---|---|
> | Channel name | `順記燒臘館` |
> | **App types** | ☑ **Web app**　←　**一定要勾** |
> | Email address | 你的 email |
>
> ※ 看到 `Linked LINE Official Account` → **留空 / 選 None**（你不需要官方帳號）

**4. 進到頻道 → 點【LIFF】頁籤 → Add**
> | 欄位 | 填什麼 |
> |---|---|
> | LIFF app name | `順記點餐` |
> | Size | **Full** |
> | **Endpoint URL** | **`https://wsad952701-beep.github.io/123/13/`** |
> | **Scopes** | ☑ **profile**　☑ **openid**　←　**兩個都要勾** |
> | Bot link feature | **Off**　←　這是官方帳號用的，關掉 |
>
> ⚠️ Endpoint URL **結尾的斜線 `/` 不能少**，要跟網址列一字不差。

**5. 複製出現的 LIFF ID**（長這樣 `2001234567-AbCdEfGh`）

**6. ★ 把頻道從 `Developing` 改成 `Published`**
> 在頻道名稱旁邊點一下就能改。
> **不改的話只有你自己登得進去，客人一律登入失敗！**

**7. 回 GitHub 編輯 `index.html 最上面的設定區`：**
```js
var LIFF_ID = '2001234567-AbCdEfGh';
```
Commit → 等 2 分鐘 → 完成。

# 為什麼訂單不能自動送進社群？

LINE 官方文件白紙黑字寫著：

> *「目前 LIFF 在 OpenChat（社群）中並未獲得官方支援，部分功能無法運作。」*

**LINE 自己不讓網頁往社群發訊息。** 這不是我們的問題，任何人都做不到。

所以最可靠的做法就是「**自動複製 → 開社群 → 客人貼上**」，約 10 秒。
送出後的畫面會有三步驟圖和一顆綠色大按鈕，客人照做就好。

> 💡 **點餐單不會提早清空** —— 一定要客人按「我已經貼上了」才清。
> 免得他還沒貼上，單子就不見了。

# 出問題怎麼辦

| 狀況 | 原因 | 怎麼修 |
|---|---|---|
| 點 LINE 鈕沒反應 | 社群連結錯了 | 檢查 `index.html 最上面的設定區` 的 `COMMUNITY_URL` |
| **（有填 LIFF_ID）登入失敗** | 頻道還是 `Developing` | 改成 `Published` |
| **（有填 LIFF_ID）「LINE 授權暫時無法使用」** | Endpoint URL 對不上 | 一字一字比對，**結尾要有 `/`** |
| 改了檔案但沒變化 | GitHub / 瀏覽器快取 | 等 2 分鐘，關掉分頁重開 |
| 客人說在社群裡開網頁怪怪的 | LINE 不支援社群裡跑 LIFF | 叫他**用瀏覽器開**（社群裡點連結 → 右上「⋯」→ 用預設瀏覽器開啟） |

# 想改東西

### 改價格 / 加菜 / 下架
改 **`index.html`**，搜尋品名（例如「順記三寶飯」），
把 `data-price="140"` 和旁邊顯示的 `140` 一起改掉。
> **加餐點面板會自動同步**，不用改第二次。

### 改營業時間
`index.html` 搜尋 `var HOURS`。數字是「從午夜起算的分鐘數」：
```js
1: [[660,810],[960,1140]],   // 週一 11:00–13:30、16:00–19:00
6: [],                       // 週六公休
```
> 660 = 11×60 → 11:00　　1140 = 19×60 → 19:00

### 讓客人先挑菜、送出時才要加入社群
`index.html` 搜尋 `LOGIN_BEFORE_ADD`，改成 `false`。

### 換社群連結
`index.html 最上面的設定區` 的 `COMMUNITY_URL`。
